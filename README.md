# Stone Paper Scissors (C++)

Das ist ein einfaches Konsolenprogramm in C++, das das Spiel Stein-Papier-Schere umsetzt.

## Beschreibung
Ich habe dieses Projekt gemacht, um grundlegende Konzepte in C++ zu üben, wie Bedingungen, Schleifen und Benutzereingaben.

Außerdem habe ich gelernt, wie man Funktionen sinnvoll verwendet und wie man verschiedene Teile des Programms miteinander verbindet.

## Funktionen
- Spielen gegen den Computer
- Zufällige Auswahl durch den Computer
- Anzeige des Ergebnisses nach jeder Runde
- Mehrere Runden möglich

## Ausführung
Projekt in Visual Studio öffnen, kompilieren und starten.
